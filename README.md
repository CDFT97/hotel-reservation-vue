# Hotel Reservation Front

## Project setup
```
npm install
```

## Enviroments vars
```
create .env with backend api url like VUE_APP_BACKEND_API=127.0.0.1:8000
```


### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production if is necesary
```
npm run build
```
